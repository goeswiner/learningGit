Abreviações e citações

Quando vamos citar códigos usando HTML podemos usar o comando code

Podemos fazer uma citação simples usando o comando q

Ou podemos fazer citações longas usando o comando blockquote assim:

Lorem ipsum dolor sit amet consectetur adipisicing elit. Dolorum voluptatum ipsam quisquam, vel quo doloribus tenetur in nam? Corrupti at numquam sint odit excepturi sunt repellat. Nesciunt vero nemo beatae.
